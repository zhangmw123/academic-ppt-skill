# Share one canonical skill across Codex and Claude Code

Academic PPT Skill will support Codex and Claude Code from one canonical skill directory containing the shared `SKILL.md`, scripts, references, assets, and bundled implementation helpers. Host-specific discovery metadata is permitted only as a thin adapter; it must not create divergent workflows, rendering behavior, quality gates, or output contracts. This prevents two independently evolving presentation-generation products while preserving native installation and invocation in both supported hosts.
