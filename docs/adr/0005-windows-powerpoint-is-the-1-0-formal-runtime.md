# Use Windows PowerPoint as the 1.0 formal runtime

Formal Academic PPT Skill 1.0 delivery is accepted only after a real render on Windows 10 or 11 with Microsoft PowerPoint. WPS is an optional separately verified target, while macOS/Linux may run the skill and create drafts but cannot claim formal visual acceptance without an Authoritative Runtime render. This chooses dependable acceptance over an unsupported cross-platform equivalence promise.
